# Apache James (apache-james)
Apache James is a portable and enterprise-ready mail server built entirely in Java. It implements SMTP, IMAP, POP3, and JMAP protocols and provides a modular architecture with a comprehensive administration REST API.

## APIs

### Apache James WebAdmin REST API
The James WebAdmin API provides REST endpoints for managing domains, users, mailboxes, mail repositories, mail queues, and server configuration, along with the JMAP protocol implementation for modern email client access.
