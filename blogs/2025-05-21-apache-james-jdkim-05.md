<p>The Apache James Project is happy to announce
the release of version 0.5 of the Apache James jDKIM library.</p>

<p>jDKIM is a DKIM implementation library written in Java. It provides
both verification and signing.</p>

<p>This release makes various libraries updates, code fixes and enhancements.</p>

<p>The Apache James PMC would like to thanks all contributors who made this release possible!</p>
