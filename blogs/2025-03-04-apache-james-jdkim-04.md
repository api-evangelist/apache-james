<p>The Apache James Project is happy to announce
the release of version 0.4 of the Apache James jDKIM library.</p>

<p>jDKIM is a DKIM implementation library written in Java. It provides
both verification and signing.</p>

<p>This release updates DNS Java to a non-vulnerable version.</p>
