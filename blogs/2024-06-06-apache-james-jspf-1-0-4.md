The Apache James developers are pleased to announce the 1.0.4 release of the JSPF library. Early adopters can download it , any issue can be reported on our issue tracker . This release fixes the DSNJava asynchronous resolver that was mis-behaving since 1.0.2 release/ The Apache James PMC would like to thanks all contributors who made this release possible!
